# Spring框架下的REST服務

使用Spring的RestController、JavaConfiguration做出一個簡單的REST服務

## 開始

這只是一個空殼專案，也可說是基礎專案，可以由此clone下來改為較正式的版本

### Prerequisites

* 需可執行gradle的套件
* tomcat7.0以上

### 安裝

下載之後轉為gradle專案

## 測試

以瀏覽器查詢http://localhost:8080/SpringRest/profileList


## 佈署

以gradle套件執行build即可，會產生war檔，即可佈署至tomcat上
